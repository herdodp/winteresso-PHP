Logout session

- digunakan untuk terputus dari session

syntax : WIN_LOGOUT("destination")

keterangan:

destination : masukkan tujuan perpindahan halaman setelah berhasil terputus dari session

Hide resubmission form notification

- digunakan agar notifikasi resubmission tidak muncul jika menekan tombol back pada browser dari halaman form yang sudah terisi dan berhasil ter submit

syntax : WIN_HIDE_STATE()

Print any

- digunakan untuk mencetak text

syntax : WINPRINT("your text")

keterangan:

your text : masukkan text
