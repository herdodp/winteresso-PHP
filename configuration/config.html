Download Library

Library dapat di unduh disini : Library winteresso

Konfigurasi :

- Download file zip library winteresso

- Extract file zip

- Copy atau pindahkan file winteresso.php ke dalam folder project

- Setup path file winteresso.php didalam file program dengan menggunakan include_once atau require_once

Download Framework

Unduh framework winteresso disini : Framework winteresso

Konfigurasi :

- Download file zip framework winteresso

- Extract file zip

- Masukkan folder hasil extract ke dalam folder htdocs web server xampp,mamp, dll.

Requirement :

- dianjurkan menggunakan php versi terbaru

- web server xampp,mamp,lamp, dll.
