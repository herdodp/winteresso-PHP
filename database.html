Connect Database

- digunakan untuk terhubung ke database

syntax : WIN_CONNECT_DB("servername", "username", "password", "database")

var global : $koneksi

keterangan:

servername : masukkan servername host

username : masukkan username host

password : masukkan password host

database : masukkan database yang digunakan

variable global '$koneksi' digunakan untuk pada parameter SQL, dll

Check connect Database

- digunakan untuk mengecek berhasil terhubung ke database atau tidak

syntax : WIN_CONNECT_DB_CHECK("servername", "username", "password", "database")

keterangan:

servername : masukkan servername host

username : masukkan username host

password : masukkan password host

database : masukkan database yang digunakan

Jika pengecekan telah selesai, hapus '_CHECK' pada syntax

Read Database Table

- digunakan untuk mengambil semua data pada table database

syntax : WINREAD_DB("first variable", "second variable", "table on database")

keterangan:

first variable : masukkan nama variable pertama (bebas)

second variable : masukkan nama variable kedua (bebas)

table on database : masukkan nama table yang akan dilakukan pengambilan data pada database

Insert to Database (username and password only for register)

- digunakan untuk memasukkan username dan password ke dalam table database

syntax : WINREAD_USERPASS("table_database", "username_field", "username_input", "password_field", "password_input")

keterangan:

table_database : masukkan nama table database

username_field : masukkan nama field untuk data username pada database

username_input : masukkan username yang akan dimasukkan kedalam database

password_field : masukkan nama field untuk data password pada database

password_input : masukkan password yang akan dimasukkan kedalam database
