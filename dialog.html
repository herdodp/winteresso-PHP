Show Dialog Onclick

- digunakan untuk menampilkan pop up alert

syntax : WIN_DIALOG_ONCLICK("your text")

keterangan:

your text : masukkan text yang ingin ditampilkan

Example : <button onclick="WIN_DIALOG_ONCLICK('ingin keluar?', 'testwinteresso.php');">keluar</button>

Show Dialog Notification

- digunakan untuk menampilkan pop up alert

syntax : WIN_DIALOG("your text")

keterangan:

your text : masukkan text yang ingin ditampilkan

Show Dialog Confirm

- digunakan untuk mengambil semua data pada table database

syntax : WIN_CONFIRM_MOVE("your text", "destination")

keterangan:

your text : masukkan text konfirmasi

destination : masukkan tujuan perpindahan halaman web jika bernilai true
